## About TOTC

TOTC by Indixpert is a comprehensive, all-in-one e-learning platform designed to empower educators and streamline the learning process. It provides a robust environment for creating, managing, and sharing online classes, digital courses, and various learning resources effortlessly. Whether you're an instructor looking to host virtual classrooms or a student eager to engage in interactive learning, TOTC offers the tools you need for success.

Our platform focuses on an interactive and efficient learning experience, combining powerful management features with a user-friendly interface.

## Features

TOTC comes packed with a wide array of features to enhance the e-learning experience:

- **Online Billing, Invoicing & Contracts:** Securely manage financial and legal transactions with customizable invoices and contracts.
- **Easy Scheduling & Attendance Tracking:** Schedule classrooms across multiple campuses and maintain detailed student attendance records.
- **Customer Tracking:** Automate email communications and organize your institution with a built-in tracking system.
- **Virtual Classroom Functionality:** Everything you can do in a physical classroom, you can do with TOTC, including managing assignments, quizzes, and exams.
- **Dedicated Instructor & Student Portals:** Tailored experiences for both educators and learners.
- **User-Friendly Interface:** A classroom-designed UI with a dedicated "Podium" space for teachers and clear visibility of student data.
- **Dynamic Teaching Tools:** Real-time assignment distribution, quizzes, and tests with automatic gradebook entry.
- **Comprehensive Class Management:** Tools for class rosters, attendance, and real-time grading.
- **One-on-One Discussions:** Private communication channels between teachers, TAs, and students within the virtual environment.
- **Interactive Course Exploration:** Easily browse and discover a variety of courses.

## Technologies Used

This project is built using modern web technologies:

- **HTML5:** For structuring the content.
- **CSS3:** For styling and layout, including a responsive design.
- **Bootstrap 5:** As the primary front-end framework for responsive design and UI components.
- **Swiper JS:** For creating interactive and touch-friendly carousels/sliders (e.g., for testimonials).
- **Google Fonts:** For custom typography (`DM Sans`, `Montserrat`, `Poppins`, `Roboto Flex`, `Nunito Sans`).
- **Bootstrap Icons & Font Awesome:** For a rich set of icons.

## Installation

To get a local copy up and running, follow these simple steps:

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/Raish0009/E-learning.git
    ```
2.  **Navigate to the project directory:**
    ```bash
    cd E-learning
    ```
3.  **Open `index.html`:**
    Simply open the `index.html` file in your web browser. All necessary CSS and JavaScript files are linked via CDNs or local paths.

This project does not require a complex build process for local viewing.

## Usage

TOTC is designed to be straightforward to use.

### For Instructors:
- Create and manage your courses.
- Upload course materials.
- Set up assignments, quizzes, and exams.
- Track student progress and attendance.

### For Students:
- Browse and enroll in courses.
- Access learning materials.
- Submit assignments and take assessments.
- Engage in private discussions with instructors.
